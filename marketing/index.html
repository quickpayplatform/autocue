How It Works

AutoQue enables remote lighting cue submission for ETC Eos consoles through a controlled workflow: Submission → Validation → Human Approval → OSC Execution → Logging.

Submit Cue

Designers submit cue data and notes from anywhere.

Validation Engine

Checks cue number > 0, levels 0–100, fade time ≥ 0, and patch ranges.

Operator Review

Operators approve or reject, preventing duplicates and locked overwrites.

OSC Execution

Approved cues translate to OSC safely—console stays local.

Logged Result

Every state change and execution is recorded for audits.

FAQ

Do cues run automatically?

No. Operator approval is required before any execution.

Will this expose our console to the internet?

No. AutoQue uses a secure VPN tunnel and a local OSC bridge.

Can we test without the venue console?

Yes. AutoQue supports ETC Nomad for offline testing.

What consoles are supported?

ETC Eos family consoles via OSC.
